# Demo

This is a demo for the progress job gem found here: [progress\_job](https://github.com/d4be4st/progress_job)

You can take a look at the demo here: [progress job demo](http://progress-job-demo.hadjic.com/)


    $ bundle install
    $ rake db:migrate
    $ rake db:seed
    $ rake jobs:work
    
    $ rake s
